# Система учета складских остатков

## Описание проекта
Проект представляет собой REST API приложение для учета складских товаров. Приложение позволяет добавлять, просматривать, редактировать и удалять товары через Swagger UI и веб-интерфейс.

## Используемые технологии
- ASP.NET Core Web API
- C#
- JSON
- HTML
- CSS
- JavaScript
- Swagger UI

## REST API методы
- GET — получение списка товаров
- POST — добавление товара
- PUT — редактирование товара
- DELETE — удаление товара

## Проверка API
Для тестирования REST API использовался Swagger UI. Через кнопку «Try it out» выполнялись запросы GET, POST, PUT и DELETE. Проверялась корректность передачи и получения данных, а также ответы сервера со статусом 200 OK.

## Веб-интерфейс
Интерфейс выполнен в современном стиле со светлым градиентным фоном. Главная страница содержит таблицу товаров, поля ввода и кнопку «Добавить». Для улучшения внешнего вида были применены CSS-стили, эффекты наведения и подсветка кнопок.

## Запуск проекта
1. Запустить проект через Visual Studio.
2. Swagger UI откроется автоматически для проверки REST API.

## Хранение данных
Данные приложения сохраняются в JSON-файл requests.json.
